# Chatbot Inteligente

Projeto desenvolvido para a faculdade utilizando Python, Flask, SQLite e Machine Learning básico.

## Objetivo
Criar um chatbot com interface web capaz de responder perguntas, armazenar histórico de conversas e integrar informações em banco de dados.

---

## Tecnologias utilizadas
- Python
- Flask
- SQLite
- Scikit-learn
- HTML/CSS
- JSON

---

## Funcionalidades
- Chatbot com respostas automáticas
- Interface web interativa
- Histórico de conversas
- Integração com banco de dados
- CRUD de informações
- Organização modular do sistema

---

## Estrutura do projeto

```bash
app.py
chatbot.py
crud.py
conexao.py
tabela.py
consultar_banco.py
ver_conversas.py
respostas.json
templates/index.html
```

---

## Como executar

Instale as bibliotecas:

```bash
pip install flask scikit-learn
```

Execute o projeto:

```bash
python app.py
```

Depois acesse no navegador:

```bash
http://127.0.0.1:5000
```
