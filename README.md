# Chatbot Inteligente

Projeto desenvolvido utilizando Python, Flask, SQLite e Machine Learning básico para simular um atendimento automatizado com armazenamento de histórico de conversas.

## 📌 Objetivo

Desenvolver um chatbot com interface web capaz de responder perguntas automaticamente, armazenar histórico de conversas e integrar informações em banco de dados.

## 🚀 Tecnologias utilizadas

* Python
* Flask
* SQLite
* Scikit-learn
* HTML/CSS
* JSON

## ⚙️ Funcionalidades

* Respostas automáticas
* Interface web interativa
* Histórico de conversas
* Integração com banco de dados
* Operações CRUD
* Organização modular do sistema

## 📁 Estrutura do projeto

* app.py
* chatbot.py
* crud.py
* conexao.py
* tabela.py
* consultar_banco.py
* ver_conversas.py
* respostas.json
* templates/index.html

## ▶️ Como executar

### Instale as bibliotecas:

```bash
pip install flask scikit-learn
```

### Execute o projeto:

```bash
python app.py
```

### Acesse no navegador:

```bash
http://127.0.0.1:5000
```
